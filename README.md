# CS2-MiniRank-Postgres
This is a basic PostgreSQL rank plugin for Counter-Strike 2 (CS2) built using the CounterStrikeSharp library.

- **Track**: 
  - Player statistics:
    - **Kills**
    - **Deaths**
    - **Assists**
    - **Flash Assists**
    - **Headshots (HS)**
    - **No-Scopes (NS)**
    - **HE Kills**
    - **Molotov Kills**
    - **Playtime**
    - **LastConnected**
    - **K/D**
